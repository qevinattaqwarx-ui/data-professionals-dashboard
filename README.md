# Data Professionals Dashboard

## Overview
This project analyzes survey data from data professionals around the world using Power BI. The dashboard provides insights into salaries, job satisfaction, programming language preferences, and career trends.


## Tools Used
- Power BI
- Power Query
- Microsoft Excel

## Dataset
Global Data Professionals Survey dataset.

## Dashboard Insights
- Salary distribution across different job roles
- Work-life balance and job satisfaction analysis
- Preferred programming languages among data professionals
- Career and industry trends
